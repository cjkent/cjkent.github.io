Osiris

Osiris is a library for building serverless web applications on AWS, with a focus on simplicity and ease of use. Its key features are:

  • A simple, familiar DSL for defining the application endpoints
  • A local server for easier development and debugging
  • Plugins for Maven and Gradle to:
    • Create the initial project
    • Build the project, generating all the necessary AWS configuration
    • Deploy the project to AWS
